¡Bienvenido a AmbientLab!

La aplicación web AmbientLab es un prototipo diseñado para replicar la estructura y funcionalidad de un Sistema de Gestión de Información de Laboratorio (LIMS, por sus siglas en inglés "Laboratory Information Management System") en formato de plataforma web.
